# 💻 COVID19-Chart

### 자바스크립트로 작성 된 COVID19-Chart📈를 TypeScript로 점진적 적용 프로젝트

<br />

## 🖐 자바스크립트 코드에 타입스크립트를 적용할 때 주의해야 할 점

- 기능적인 변경은 절대 하지 않을 것
- 테스트 커버리지가 낮을 땐 함부로 타입스크립트를 적용하지 않을 것
- 처음부터 타입을 엄격하게 적용하지 않을 것 (점진적으로 strict 레벨을 증가)
